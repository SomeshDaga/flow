struct flow\+::\+Polling\+Lock$<$ Basic\+Lockable\+T $>$}

Stand-\/in type used to signify that captors will be used in a threaded context, but will not wait for data. 

Captors use {\ttfamily Basic\+LockableT} to protect data input/output from the capture queue, but do not wait on a condition variable for new inputs before attempting to run a synchronization policy. This allows for polling with the {\ttfamily \mbox{\hyperlink{classflow_1_1_captor_interface_ab645172a3401cc978fd4618a64a83e3d}{Captor\+::capture}}} method ~\newline
