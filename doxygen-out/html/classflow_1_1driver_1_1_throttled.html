Detailed Description

template<typename DispatchT, typename LockPolicyT = NoLock, typename ContainerT = DefaultContainer<DispatchT>, typename QueueMonitorT = DefaultDispatchQueueMonitor>
class flow::driver::Throttled< DispatchT, LockPolicyT, ContainerT, QueueMonitorT >

Throttled next element driving capture object.

Captures the next oldest data element, limited to a max expected period. This means that some elements are skipped if the input rate indicated by data sequence stamps is higher than the throttled rate.
Establishes a sequencing range with range.lower_stamp == range.upper_stamp equal to the captured element stamp. Removes captured element from buffer.
