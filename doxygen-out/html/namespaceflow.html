Detailed Description

Flow input synchronization library components.

Implements the C++17 utility std::integer_sequence and related compile-time utility objects.

Defines a utility similar to std::apply (C++14)

Enumeration Type Documentation

◆ State

enum flow::State : int
strong

Evaluated Captor state.

These are used by captors to direct internal data capture behavior

Enumerator
RETRY 

Captor should continue waiting for data after prime attempt.

PRIMED 

Captor has captured data and its ready.

ABORT 

Captor has requested to abort current capture attempt.

TIMEOUT 

Captor has hit a data-wait timeout.

ERROR_DRIVER_LOWER_BOUND_EXCEEDED 

Error code used to indicate that driving violated external lower bound.

SKIP_FRAME_QUEUE_PRECONDITION 

Code use to indicate that the current sync frame is to be skipped, namely when when a queue monitor object is used to precondition capture

_N_STATES 

Total number of captor states.
